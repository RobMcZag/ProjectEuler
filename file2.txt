Ibisense has integrated varying data sources to cloud services. The sources
have ranged from battery operated embedded sensors to industrial plants.
When working with industrial data, the security is always a concern. We
have the methods to gather data without compromising the security of the
operations.

Big Data storage

Industrial internet data sources can generate huge amounts of complex data.
We apply the industry standard Big Data storage tools, like Hadoop and Cassandra.
In the field of data storage, our core expertise lies in efficiently storing
vast amounts of Time Series Data. 

Realtime analytics
Turning Big Data to knowledge and more importantly actions is a crucial part
of the Industrial Internet. Computers and statistical tools are excellent in
finding features from Big Data. Some of these features and correlations are
meaningful, but most are artificial one. It soon becomes a necessity to
include human insight into the analytics process.

Ibisense offers several levels of analytics integration tools. With these
tools, it is possible to add automatically executed analytics tools
to the cloud. The tools allow domain experts to create simple automatically
run analyses without the need of programming experts. On the other end of
the spectrum, Ibisense offers tools for the expert data analysts working with
tools like Matlab, SPSS, Python, Hadoop etc.
